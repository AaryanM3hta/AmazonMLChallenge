# Dataset Not Included

The dataset used in this project is **not uploaded to GitHub** because:

- It is too large for version control  
- It is part of the Amazon ML Challenge dataset  
- Uploading it may violate data usage policies  

To run this project, please download the dataset from the **Amazon ML Challenge portal** and place the required files in this folder.

Example structure:

data/
train.csv
test.csv
images/
